# Ideas

아이디어만 넣는 공간이다. 여기서는 아직 실행, 검증, 구현을 하지 않는다.

## How To Use

새 아이디어는 짧게만 적는다.

```text
아이디어: {한 줄 설명}
왜 떠올랐나: {상황/문제}
타깃: {생각나는 사용자}
```

Codex는 이 공간에서 다음만 한다.

- 아이디어를 잃어버리지 않게 정리
- 비슷한 아이디어끼리 묶기
- 실행 후보가 될 만한 것만 Command Center 또는 프로젝트로 승격 제안

## Promotion Rule

아이디어가 다음 중 하나를 만족하면 프로젝트 후보로 승격한다.

- 사용자가 `이걸로 하자`라고 말함
- 특정 챌린지/제출처가 생김
- MVP 기능 3개 이하로 줄일 수 있음
- 사용자 결정이 필요한 항목이 명확해짐
