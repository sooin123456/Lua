# Prior art search rules

## 검색 범위

- KIPRIS(한국) — 필수
- USPTO(미국) — 필수
- EPO(유럽) — 권장
- 학술(Google Scholar, IEEE Xplore 등) — 권장

## 검색 키워드 전략

- 기술 용어(한/영 동의어)
- 기능·효과 표현(해결 과제 중심)
- IPC/FI/FT 분류
- 주요 출원인(경쟁사) 모니터링

## 검증 횟수

- 최소 3회 서로 다른 키워드 조합
- 각 검색마다 상위 후보 20건을 스크리닝

## 출력 형식 (선행 1건당)

- 출원번호 / 공개번호 / 등록번호
- 출원일 / 공개일 / 등록일
- 출원인
- 요지(한 단락)
- 본 발명과의 차이(한 단락)

## 충돌 처리

- 유사도가 높으면 청구 범위를 조정하는 쪽으로 기록
- “없다”고 단정하지 말고 검색식·날짜를 남긴다

## 금지

- 비공개 정보를 인용
- 웹 블로그만으로 선행 부정
- 번역기만으로 해외 특허 요지 확정
