# Grant programs reference

> **중요**: 정부 R&D 과제 양식·정책은 매년 변경된다. 분기마다 `last_verified`를 갱신한다.  
> 실제로 신청하는 program부터 표를 채운다.

## KEIT (한국산업기술기획평가원)

| 항목 | 내용 |
|---|---|
| 관할 부처 | 산업통상자원부 |
| 주요 사업 | 산업기술혁신사업, 소재부품장비, 한국형 뉴딜(세부 과제군은 공고별 상이) |
| 평가 가중치 | 기술성 40% / 사업성 30% / 수행능력 30% (program·공고별 상이 — 공고 PDF 기준) |
| 양식 분량 | 본문 통상 25–40p (첨부별도) |
| 마감 주기 | 통상 연 2–3회(상반기·하반기) — 공고문 확인 |
| 핵심 정책 키워드 | 산업 디지털 전환, 친환경·탄소저감, 소부장, 제조 안전 |
| URL | https://www.keit.re.kr |
| 너의 노트 | EMS·에너지 모니터링 실증 중심으로 기술성 근거를 쌓는다. 사업성은 국내 B2G 레퍼런스와 수출 가능성을 짧게 연결. 평가위원은 “측정 재현성·로그”를 자주 묻는다. |
