## 신뢰도 등급

| 등급 | 출처 유형 | 사용 가능? |
|---|---|---|
| A | 1차 자료 (vendor docs, SEC, gov, peer-reviewed) | 단정조 OK |
| B | 1차 인용한 established 매체 (NYT, FT, IEEE, 한겨레, 매경) | “보도에 따르면” 형태로 |
| C | 분석가 보고서 (Gartner, McKinsey, 한국에너지경제연구원) | 출처와 주체 명시 필수 |
| D | 블로그·중간 매체 (분야 전문가) | “한 의견으로” 표지 |
| E | 익명 / 콘텐츠 팜 / SEO 글 | 사용 X |

## 시점 검증

- 가격/스펙: 6개월 이내
- 트렌드: 12개월 이내
- 기본 원리·역사: 시점 무관
- 정부 정책: 최신 개정 확인 필수
- 모든 인용에 게시 날짜를 적는다

## Conflict resolution

1. 가장 최신 + 가장 1차에 가까운 자료를 우선한다.
2. 충돌을 합치지 말고 “A는 X, B는 Y”로 병기한다.
3. 한쪽으로만 뭉개지 않는다.

## Red flag 출처

- 게시일이 없다
- 작성자 정보가 없다
- “AI generated” 라벨 없이 기계적으로만 읽히는 글
- affiliate 링크가 과도하다
