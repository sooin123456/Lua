# Decision Board

사용자가 직접 골라야 하는 것만 모으는 공간이다.

Codex는 구현, 조사, 정리, 초안 작성, 검증은 자동으로 진행하되, 아래 성격의 항목은 여기로 올린다.

- 브랜드명/표기
- 대표 이미지/스크린샷
- 제출 전 최종 방향
- 비용, 계정, API key, 외부 발행
- 되돌리기 어려운 공개/공유/배포
