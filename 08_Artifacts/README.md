# Artifacts

This folder is the lightweight artifact layer for Lua.

Use it before adopting a heavier artifact service. The rule is simple: if an output may be reused, shared, reviewed, shipped, or cited later, it needs a ledger entry.

## Use For

- app builds and demo URLs
- proposal drafts
- patent drafts
- datasets and exports
- research reports
- screenshots
- skill bundles and prompt packs

## Do Not Store Here

- secrets
- API keys
- private tokens
- large binaries that belong in GitHub releases, cloud storage, or a package registry

## Command

Use `/artifact-log {name}`.
